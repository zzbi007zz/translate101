🔧 Debugging Tips

If translation doesn't work:

  1. Open Chrome DevTools (F12)
  2. Check Console tab for errors
  3. Look for "Instant Translate content script loaded" message
  4. Verify extension is enabled in chrome://extensions
  5. Make sure API key is configured in extension popup
  6. Check Network tab for API calls when translation is triggered

Expected Console Output:

Instant Translate content script loaded
Requesting translation...
Translation received: [translated text]
Latency: [XX] ms
